 AliviaTech

--- Descripción del Proyecto
AliviaTech es una solución digital enfocada en el bienestar emocional. Integra una aplicación web, una app móvil y un wearable inteligente para monitorear señales fisiológicas y brindar recomendaciones personalizadas.
